# Prerequisits
  * Install the Mongo DB [https://docs.mongodb.com/manual/tutorial/install-mongodb-on-windows-unattended/]
  * node

# Setup
* Clone the Repository
* Checkout the "usercursorsupgrade" branch
* npm install

# Run server
* cd server
* node server-main.js

# Run Client
* node clientserver.js
